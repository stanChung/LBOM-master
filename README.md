# LBOM(訂餐系統)
![ScreenShot](https://github.com/stanChung/LBOM/blob/master/LBOM_H.PNG)
  
### **功能描述：**
> - 可維護商家相關資料
> - 可維護訂購相關資料
> - 將訂購資料匯出為EXCEL或產生報表

### **使用工具與技術：**   
> - ASP.NET MVC
> - EasyUI
> - Devexpress Reporting
> - Oracle
